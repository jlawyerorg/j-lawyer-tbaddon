Latest changes

  • Version 1.9.7
    • Added English and German localization for the manifest, popups, options, calendar dialog, message-assignment dialog, image editing, context menus and update page.
    • Adjusted the add-on name to the allowed format "j-lawyer.org for Thunderbird".
    • Removed unused activeTab and tabs permissions and moved broad server access to an optional host permission.
    • Migrated Thunderbird message tag handling to the current messages.tags API.
    • Fixed the asynchronous runtime message listener in the message popup.
    • Updated the build package so local agent files are not included in the XPI.
  • Version 1.9.5
    • Fixed: The activity log no longer uses unsafe HTML writes and is readable again in the light layout.
  • Version 1.9.4
    • Fixed: Creation dates for saved emails now reuse existing document metadata and avoid invalid update payloads.
    • Fixed: OCR for newly saved attachments no longer fails because of an internal variable reference.
  • Version 1.9.3
    • Image attachments can now be reordered in the edit dialog.
    • Fixed: After changing the displayed message, image editing uses the attachment of the currently selected message.
  • Version 1.9.x and earlier
    • Added rename templates, improved case lookup and added a combined message and attachment save action.
    • Improved popup layout, window sizing, folder handling, attachment processing and image editing.
    • Added OCR options, document tagging, context menu fixes, compose-window case detection and email template insertion.
    • Added calendar integration, logging, automatic data updates and multiple performance and stability fixes.
